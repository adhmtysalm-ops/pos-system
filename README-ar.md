## 🌟 المميزات الرئيسية

### 💻 شاشة الكاشير (POS)
- **واجهة سريعة الاستجابة**: مصممة لضمان أسرع عملية بيع للعملاء دون تأخير.
- **دعم الباركود**: إضافة المنتجات للسلة فوراً باستخدام قارئ الباركود.
- **الدرج العائم للموبايل**: عند فتح النظام من الجوال، تظهر سلة المشتريات كدرج سحابي أنيق يمكن فتحه بـ (الزر العائم) لتوفير أقصى مساحة لرؤية المنتجات.
- **الطباعة الحرارية**: طباعة الفواتير مباشرة على طابعات الريسيت (80mm) من المتصفح.

### 📦 إدارة المخازن والمنتجات
- **تتبع حي للمخزون**: يمنع النظام بيع أي منتج كميته صفر لضمان دقة الجرد.
- **صور المنتجات**: إمكانية رفع صور ديناميكية لكل منتج يتم حفظها وعرضها محلياً بأمان.
- **التصنيفات الملونة**: تنظيم المنتجات في أقسام مع تمييزها بألوان مخصصة لسرعة الوصول.

### 👥 نظام الصلاحيات
- **المدير (Admin)**: تحكم كامل في كل شاشات النظام، الموظفين، التقارير المالية، الإعدادات، وحذف السجلات.
- **الكاشير (Cashier)**: صلاحيات محدودة تقتصر على شاشة البيع وتأكيد حضوره وانصرافه فقط. يمكن للمدير تحديد "الحد الأقصى للخصم المسموح به" لكل كاشير.

### 💼 إدارة العملاء والديون
- تسجيل بيانات العملاء وحفظها.
- إمكانية (البيع الآجل)، وتتبع ديون العملاء وتسجيل الدفعات الجزئية.
- فلترة بضغطة زر لرؤية "العملاء الذين عليهم ديون فقط".

### 📊 التقارير والإحصائيات
- **لوحة التحكم (Dashboard)**: رسومات بيانية حية توضح حركة المبيعات في آخر 7 أيام.
- تقارير مفصلة للمبيعات اليومية، والمصروفات، والمشتريات. حساب دقيق لصافي الربح.

### 🏢 الإدارة والعمليات (HR)
- **الموظفون**: تسجيل الموظفين وتحديد رواتبهم وبياناتهم.
- **الحضور والانصراف**: نظام لتسجيل ساعات العمل اليومية لكل موظف.
- **الموردون والمشتريات**: تتبع حسابات الموردين وإضافة بضاعة جديدة للمخزن.
- **المصروفات**: تسجيل كافة مصروفات المحل (إيجار، كهرباء، صيانة).

## 🚀 التقنيات المستخدمة (Architecture)

- **الواجهة الأمامية**: React (Vite), React Router v6, TailwindCSS, Lucide Icons.
- **الواجهة الخلفية**: Node.js, Express, `mysql2/promise` لاستعلامات قواعد بيانات فائقة السرعة، JWT للتوثيق والحماية.
- **قاعدة البيانات**: MariaDB / MySQL. بنية مترابطة (Relational) لضمان عدم تضارب البيانات.
- **التجاوب الكامل (Responsive)**: النظام يستخدم تقنية `dvh` ليظهر بشكل مثالي على متصفحات الجوال دون أن تختفي الأزرار السفلية.
- **العمل المحلي المستقل**: لضمان أقصى حماية، النظام يعمل محلياً فقط (Localhost) ولا يمكن الوصول إليه من أجهزة أخرى عبر الشبكة.

## ⚡ التثبيت والتشغيل بضغطة زر

لقد صممنا عملية التثبيت لتكون أسهل ما يمكن بفضل سكريبت التشغيل التلقائي.

### المتطلبات المسبقة
- Node.js (إصدار 16 فما فوق)
- XAMPP أو أي سيرفر MySQL/MariaDB يعمل على بورت `3306`.

### خطوات التثبيت والتشغيل (Step-by-Step)

#### 1. تجهيز قاعدة البيانات
قبل البدء، تأكد من تشغيل سيرفر MySQL (من خلال XAMPP مثلاً)، ثم قم بإنشاء قاعدة بيانات فارغة باسم `POS`.
- **طريقة 1 (عبر سطر الأوامر):**
  ```bash
  mysql -u root -e "CREATE DATABASE IF NOT EXISTS POS CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"
  ```
- **طريقة 2 (عبر phpMyAdmin):**
  1. افتح المتصفح واذهب إلى `http://localhost/phpmyadmin`
  2. اضغط على **Databases** (قواعد البيانات).
  3. اكتب `POS` واضغط **Create** (إنشاء).

#### 2. تثبيت النظام
1. قم بتحميل المشروع:
   ```bash
   git clone https://github.com/adhmtysalm-ops/pos-system.git
   cd pos-system
   ```
2. أعطِ تصريح التشغيل للسكريبتات ثم شغل سكريبت التثبيت:
   ```bash
   chmod +x setup.sh start.sh
   ./setup.sh
   ```

**ما الذي يفعله سكريبت `./setup.sh` تلقائياً؟**
1. تحميل وتنصيب مكتبات الباك اند والفرونت اند.
2. زراعة الجداول في قاعدة البيانات (`schema.sql`) وإدخال بيانات تجريبية وحساب المدير التلقائي (`seed.js`).
3. تجهيز ملفات البيئة (`.env`) تلقائياً.

### التشغيل اليومي للنظام
في كل مرة تريد فيها فتح النظام للعمل، فقط قم بتشغيل:
```bash
./start.sh
```
هذا السكريبت سيقوم بتشغيل سيرفرات النظام فوراً وبسرعة فائقة دون إعادة التثبيت أو تكرار البيانات التجريبية.

## 🔒 العمل بشكل محلي (Localhost)

لأسباب أمنية ولتجنب أية تعارضات في الشبكة، النظام مهيأ الآن للعمل حصرياً على نفس الجهاز المُشغل للسيرفر.

- **كيفية الدخول**: افتح المتصفح من نفس الكمبيوتر واكتب `http://localhost:5173`.
- **ملاحظة**: النظام محمي ضد الاتصالات الخارجية، ولن يعمل إذا حاولت الدخول إليه من هاتف أو جهاز آخر عبر الشبكة.
